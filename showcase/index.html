The Project

Label-Liar 2.0 is a cutting-edge Hybrid AI application designed to demystify complex food ingredient labels. By combining OCR technology with the power of Google Gemini Pro, it provides personalized health insights tailored to your specific dietary needs and profiles, such as "Diabetic", "Vegan", or "Allergy-prone".

Key Features

OCR Ingredient Scanning

Powered by Google ML Kit to accurately read even the most complex labels instantly.

AI Health Auditor

Uses Google Gemini Pro to interpret scans and provide deep nutritional analysis.

Personalized Profiles

Custom logic for various health conditions like Diabetes, Vegan diets, and more.

Premium UI/UX

A modern glassmorphism interface that makes health tracking a beautiful experience.

The Tech Stack

Frontend

Flutter

Cross-platform mobile application optimized for Android with custom Dart services.

Backend

Node.js & Express

Robust server handling API requests, Gemini integrations, and MongoDB connections.

Intelligence

Google Gemini Pro

State-of-the-art LLM for intelligent interpretation of food ingredients.

Database

MongoDB

Scalable NoSQL database for managing user profiles and audit history.

How to Use

1

Setup Backend

cd backend
npm install
npm start

Server starts at http://localhost:5000

2

Run Frontend

cd frontend
flutter pub get
flutter run

Optimized for Android mobile devices.
